# Книжный интернет-магазин

## Автор

Гинжул Мария

## Постановка задачи

Требуется разработать консольное приложение на C#, имитирующее работу книжного интернет-магазина. Приложение решает прикладную задачу предоставления пользователю возможности выбора способа оплаты при оформлении заказа. Приложение должно позволять пользователю просматривать доступные книги, формировать заказ и оплачивать его одним из предложенных способов.

## Использованный паттерн: Strategy

### Цель применения
Strategy — поведенческий паттерн проектирования, который определяет семейство алгоритмов, инкапсулирует каждый из них и делает их взаимозаменяемыми. Паттерн позволяет передавать алгоритм в контекст как объект и менять его во время выполнения программы.
В данном проекте паттерн Strategy используется для реализации различных способов оплаты.

Каждый способ оплаты вынесен в отдельный класс, реализующий общий интерфейс. Контекст просто делегирует выполнение оплаты текущей стратегии, абстрагируясь от деталей реализации конкретного способа.
